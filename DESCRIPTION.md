# Post Install System

## Overview
The Post Install System is a tool designed to automate the configuration and setup of servers after the initial operating system installation. It simplifies the process by generating and executing bash scripts on remote servers.

## Key Features

- **Script Generation**: Dynamically generates bash scripts tailored to specific configuration needs.
- **Remote Execution**: Connects to remote servers to execute the generated post-installation scripts.
- **Automation**: Streamlines repetitive tasks such as system updates, package installations, and basic configuration.
- **Interactive Interface**: Provides a user-friendly way to select configuration options.

## Usage
The system is intended to be used by system administrators to quickly bring new servers to a desired state.
